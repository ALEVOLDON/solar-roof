# Solaris: Программная интеграция и визуализация солнечной кровли

Интерактивная 3D-презентация и автоматизированная визуализация плоской бетонной крыши с оптимизированной сеткой солнечных панелей. 

Проект разработан программно с использованием Python API для Blender, анимирован для демонстрации клиенту и оформлен в виде премиального веб-лендинга.

---

## 📋 Описание проекта

Проект демонстрирует концепцию размещения фотоэлектрических систем на плоской кровле здания:
*   **Бетонная плита крыши**: 9.66м x 6.23м x 0.3м (общая площадь ~60.18 м², периметр ~31.78м).
*   **Сетка солнечных панелей**: 15 кремниевых панелей размером 1.70м x 1.00м x 0.04м в конфигурации 5x3.
*   **Ориентация**: Угол наклона 15 градусов с направлением строго на Юг для максимальной энергоэффективности.
*   **Материалы**: Процедурные PBR-материалы (матовый бетон, анодированный алюминий, оцинкованная сталь и кремниевые ячейки с токопроводящей сеткой).

---

## ⚡ Особенности и функционал

### 1. Автоматическое 3D-моделирование и Анимация (`generate_solar_roof.py`)
Скрипт на Python полностью воссоздает сцену в Blender «с нуля» и настраивает 12-секундную (360 кадров при 30 FPS) анимацию, состоящую из трех последовательных этапов:
1.  **Кадры 1–120 (Поэтапная сборка):** Плавное масштабирование крыши и последовательное появление опорных ферм и панелей в шахматном порядке.
2.  **Кадры 120–240 (Облет камеры):** Круговой облет камеры на 360° вокруг центра крыши для демонстрации конструкции со всех ракурсов.
3.  **Кадры 240–360 (Движение солнца):** Анализ теней от рассвета до заката с динамическим изменением интенсивности и теплоты освещения.

### 2. Оптимизация рендеринга (под NVIDIA GeForce RTX 3050)
Скрипт автоматически оптимизирует настройки рендера Cycles для видеокарт серии RTX:
*   Аппаратное ускорение **NVIDIA OptiX** (использование RT-ядер).
*   Адаптивное сэмплирование (с порогом шума **0.05** и лимитом в **256 сэмплов** вместо дефолтных 4096).
*   Шумоподавление **OpenImageDenoise** на GPU.
*   *Результат:* Время рендеринга сокращено в 10–15 раз при сохранении фотореалистичного качества.

### 3. Интерактивный клиентский лендинг (`index.html`)
Сайт-презентация в темных тонах (соответствует цветовой гамме слайдов), разработанный без тяжелых фреймворков (чистый HTML/CSS/JS):
*   **Интерактивный плеер:** Кнопки этапов анимации позволяют мгновенно перематывать видео на нужный фрагмент (Сборка, Облет, Движение солнца).
*   **Карусель слайдов:** Удобный просмотр 12 слайдов презентации прямо на сайте.
*   **Ключевые показатели:** Визуальные карточки характеристик площади и углов наклона.
*   **Адаптивность:** Полная отзывчивость интерфейса, включая аккуратное бургер-меню для мобильных устройств.

## 🚀 Инструкция по запуску

### Запуск генерации в Blender
1. Откройте Blender.
2. Перейдите на вкладку **Scripting** и откройте файл `generate_solar_roof.py`.
3. Нажмите кнопку **Run Script** (или запустите из консоли: `blender --background --python generate_solar_roof.py`).
4. Нажмите **Пробел** на таймлайне для воспроизведения анимации.

### Просмотр лендинга
Просто откройте файл `index.html` двойным кликом в любом браузере.

### Публикация в сети (GitHub Pages)
1. Выполните в терминале вход в свой аккаунт:
   ```bash
   gh auth login
   ```
2. Создайте репозиторий и загрузите файлы:
   ```bash
   gh repo create solar-roof --public --source=. --push
   ```
3. Перейдите в настройки репозитория на GitHub: **Settings > Pages**.
4. В разделе **Build and deployment** выберите ветку `master` (или `main`) и нажмите **Save**. Сайт будет доступен по ссылке через пару минут!
